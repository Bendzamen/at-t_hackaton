# Langflow Flows

This folder contains exported Langflow flows in JSON format.

## How to Import

1. Start the Docker container:
   ```bash
   docker-compose up -d
   ```

2. Open Langflow at http://localhost:7860

3. Click "New Flow" or the import icon

4. Select "Import from JSON"

5. Upload the JSON file from this folder

## Available Flows

- Add your flow descriptions here as you export them
- Example: `agent-with-tools.json` - Agent with file system and shell tools
